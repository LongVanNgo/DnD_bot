# DnD_bot
Discord Bot to ease the DnD gameplay during covid pandemic.

## Dependencies
```
pip install discord.py python-dotenv
```


## Commands
Bot can be called with commands in Discord channel

### Help
```
_help
```
### Roll
Roll x dice with y dots
```
_roll xdy
```
####Example
```
_roll 4d6
```
